# Gerenciador de Supermercado

Integrantes: Marianne Fisch e Henrique Zotti


C# é uma linguagem de programação orientada a objetos desenvolvida pela Microsoft em 2000. Ela foi criada para facilitar o desenvolvimento de aplicações modernas, sendo amplamente utilizada na criação de sistemas desktop, aplicações web, jogos e aplicativos móveis.

A linguagem faz parte da plataforma .NET e se destaca por sua sintaxe simples, segurança, alto desempenho e facilidade de manutenção. Por ser fortemente tipada e orientada a objetos, o C# ajuda os desenvolvedores a criar programas organizados e confiáveis.

Como compilar e executar um programa em C#
Instale o SDK do .NET no computador.
Crie um arquivo chamado Program.cs.
Escreva o código-fonte.
Abra o terminal na pasta do projeto.
Compile e execute o programa com o comando: dotnet run
